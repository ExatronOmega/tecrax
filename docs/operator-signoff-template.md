# Sanitized operator sign-off template

- Date and repository HEADs
- Package/profile version and profile digest
- Intent id, read-only mode and generic target role
- Connector backend class
- GovEngine verdict and policy digest
- Receipt, validation and SCLite bundle digests
- Coverage, assessment and explicit non-claims
- Secret/private-topology scan result
- Run class: fixture, lab or operator-verified

Do not include IP addresses, hostnames, usernames, key/known-hosts paths, local ports, tokens,
passwords, certificates, peer addresses, private operation ids, raw outputs or private
environment files.
